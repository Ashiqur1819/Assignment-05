Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from the DOM?

To select elements from the DOM:


  • By ID: document.getElementById('myId');
  • By Class: document.getElementsByClassName('myClass');
  • By Tag: document.getElementsByClassName('div');
  • By CSS Selector: document.querySelector('.myClass'); / document.querySelectorAll('.myClass');
  • By Name: document.getElementsByName('myElementName');
What is the event delegation in the context of the DOM, and why is it useful?

Event delegation is a design pattern in JavaScript used to efficiently manage and handle events on multiple child elements by attaching a single event listener to a common ancestor element.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes and styles using JavaScript by selecting the element and then using various properties and methods. Here's how to do it:


Manipulating Attributes:

  • Get an Attribute: element.getAttribute('attributeName');
  • Set an Attribute: element.setAttribute('attributeName', 'value');
  • Remove an Attribute: element.removeAttribute('attributeName');

Manipulating Styles:

  • Set Inline Styles: element.style.propertyName = 'value';
  • Remove Inline Styles: element.style.propertyName = 'value';

Adding/Removing CSS Classes:

  • Add a Class: element.classList.add('className');
  • Remove a Class: element.classList.remove('className');
  • Toggle a Class: element.classList.toggle('className');
